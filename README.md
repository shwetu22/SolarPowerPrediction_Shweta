# Solar Power Prediction System

ML pipeline that predicts solar energy output using weather data.

## Setup

pip install -r requirements.txt

## Steps

1 Generate dataset
python generate_dataset.py

2 Train model
python train_model.py

3 Fetch forecast
python weather_api.py

4 Predict solar output
python predict.py

## Model
RandomForestRegressor
